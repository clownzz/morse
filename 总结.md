## 二.软件部分
软件部分主要通过在arduino软件进行实现，在第一次作业，我们通过库函数来实现morse电码的转换
morse.cpp
    #include "Arduino.h"
    #include "Morse.h"
    
    Morse::Morse(int pin)
    {
    	pinMode(pin,OUTPUT);
    	_pin=pin;
    }
    
    void Morse::dot()
    {
    	digitalWrite(_pin,HIGH);
    	delay(250);
    	digitalWrite(_pin,LOW);
    	delay(250);
    }
    
    void Morse::dash()
    {
    	digitalWrite(_pin,HIGH);
    	delay(1000);
    	digitalWrite(_pin,LOW);
    	delay(250);
    }
morse.ino
